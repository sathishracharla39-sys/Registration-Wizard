# Registration Wizard

A multi-step registration form built using React, React Hook Form, and Zod.

# Live Demo
https://registration-wizard-coral.vercel.app/

## Features

- Multi-Step Form
- Real-Time Validation
- Progress Bar
- Show/Hide Password
- State Persistence
- Review Screen
- Success Message
- React Hook Form
- Zod Validation

## Technologies

- React
- Vite
- React Hook Form
- Zod
- CSS3

## Setup

npm install

npm run dev
